## 📐 汎用テンプレート構造

すべての商材に使える基本構造：

```
[問題キャラクター] + [解決手段] + [結果の変化]
[特殊POV視点] + [環境詳細]
[3Dレンダリングスタイル] + [品質タグ]
```

### 記入例
```
[Stubborn dirt character] + [Product cleaning action] + [Transformation to clean]
[POV from surface being cleaned] + [Realistic material texture]
[3D rendered in satisfying cleaning style] + [octane render, 8k]
```

## 🎯 商材別カスタマイズポイント

### 口腔ケア
- **視点**: 口の中、歯の表面、歯茎
- **敵キャラ**: バクテリア、歯垢、食べかす
- **色**: 清潔感のある青/白、汚れは黄色/緑

### ヘアケア
- **視点**: 頭皮下、毛根、毛穴
- **敵キャラ**: 皮脂、汚れ、弱った毛根
- **色**: 健康的なピンク、栄養の黄金色

### スキンケア
- **視点**: 毛穴の中、角質層、真皮層
- **敵キャラ**: 角栓、乾燥、角質
- **色**: 健康な肌のピンク、保湿の青

### 健康食品
- **視点**: 腸内、血管内、細胞内
- **敵キャラ**: 悪玉菌、毒素、活性酸素
- **色**: 健康的な赤/ピンク、栄養の緑/黄金

### 洗剤
- **視点**: 汚れている表面、繊維の奥
- **敵キャラ**: 油汚れ、カビ、バクテリア
- **色**: 汚れの茶色/黒、清潔の白/青
