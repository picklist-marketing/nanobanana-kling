# 🦠 グロテスク・コメディスタイル（参考動画風）

参考動画（Higgsfield Assets）のような、口腔内視点のグロテスクかつコミカルな3Dキャラクター作成用プロンプト集

---

## 🎯 このスタイルの核心要素

1. **極端なPOV（視点）**: 口の中、喉の中など、通常ありえない視点
2. **グロテスク × コミカル**: 気持ち悪いけど笑える絶妙なバランス
3. **ハイパーリアリスティック質感**: 映画級の3Dレンダリング
4. **誇張された表情**: アニメ的な大げさなリアクション
5. **生物学的環境**: 歯、舌、唾液、粘液などの有機的ディテール

## ⚠️ 注意事項

### ✅ このスタイルで効果的な要素
- **グロテスク × 可愛い** のコントラスト
- **極端なPOV** で非日常的な視点
- **ハイパーリアル環境** + **カートゥーンキャラ** の組み合わせ
- **生物学的な動き** で有機的なリアリティ
- **コメディタイミング** で不快感を緩和

### ❌ 避けるべき要素
- 過度にリアルすぎるキャラクター（不気味の谷）
- 真面目すぎる医学的描写（コメディ要素が消える）
- 静的すぎる構図（動画化で映えない）
- 不明瞭なPOV（視点の面白さが伝わらない）

## 🚀 実験的アイデア

### シリーズ化コンセプト
1. **第1話**: 口に入る瞬間
2. **第2話**: 歯の間で格闘
3. **第3話**: 舌の上でサーフィン
4. **第4話**: 喉へ落下
5. **第5話**: 脱出成功（くしゃみで外へ）

### インタラクティブ要素
- 視聴者が選択肢を選べるストーリー分岐
- 複数キャラクターの視点切り替え
- BGMと効果音で没入感アップ
